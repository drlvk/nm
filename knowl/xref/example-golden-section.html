Example 30.4.2. Minimize by golden section.

Minimize \(f(x) = x^4 - 3 x^2 + x + 1\) using the golden section method on the interval \([-2, 2]\text{.}\)
🔗
Solution.
f = @(x) x.^4 - 3*x.^2 + x + 1;
a = -2;
b = 2;
r = (3-sqrt(5))/2;

while b-a >= 100*eps(a)
    c = a + r*(b-a);
    d = b - r*(b-a);
    if f(c) < f(d)
        b = d;
    else
        a = c;
    end
end

fprintf('Found a minimum x = %g with f(x) = %g \n', c, f(c));
