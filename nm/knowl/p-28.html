We can use a vector of indexes to extract several elements of a vector or a matrix. For example, v(3:end-1) extracts all entries of v starting with the 3rd one and ending with the one before the last. If v was [8 6 4 3 2 1], the result would be [4 3 2]. For another example, v(2:2:end) extracts all even-numbered elements of v. In the above example this would be [6 3 1]. For a matrix A, we can select both rows and columns at the same time: A(2:3, 2:5) means taking the elements of A that appear in rows 2-3 and in columns 2-5. The result is a submatrix of size 2Ã—4.
